The following are steps to modify header and footer content in CIMB Gateway Web Module.

1.	Design the header and footer web content in HTML format. (Content Width: 778px, Content Height: 232px)
2.	Save and rename the header and footer web content as “pg_header.htm” and “pg_footer.htm” respectively.
	Note: The web content must be in .htm extension.
3.	Copy the web content to system header/footer folder at location: <application folder>\CIMB_WebPage. eg. D:\CIMB_Gateway_FE\ CIMB_WebPage.
4.	Copy all the resources required by the header and footer web content (egg. Pictures, flash…)to the same folder.
